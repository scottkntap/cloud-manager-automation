    Retrieve AWS private links

    08/01/2022 Contributors

    GET /account/{accountId}/providers/cloudmanager_cbs/api/v1/aws/awsAccountId/{awsAccountId}/region/{regionCode}/privateLink

    Retrieves a list of AWS private links from a specified region that belongs to a specified AWS account ID from the Cloud Manager SaaS application.

    Token usage: Cloud Manager user token.

    Parameters

    Name Type In Required Description

    accountId

    string

    path

    True

    Account ID

    regionCode

    string

    path

    True

    Region Code

    awsAccountId

    string

    path

    True

    AWS Account ID

    authorization

    string

    header

    True

    User Token

    x-agent-id

    string

    header

    True

    Request Identifier

    Response
