    Perform a volume and file-level restore

    11/11/2022 Contributors

    POST /account/{accountId}/providers/cloudmanager_cbs/api/v1/restore

    Performs a volume and file-level restore operation. This API endpoint validates whether a backup of a volume exists. Restore is not supported on data protection volumes. A restore operation is not supported on a target volume with a different language code than the source volume. A restore operation is not supported for volumes with a size less than 1 GB. Restore is not allowed on a volume enabled with a SnapLock license (WORM license). It checks whether the source and target cloud providers match. It also verifies whether the backup and restore operation is supported on the specified working environments.

    Token usage: Cloud Manager user token.
